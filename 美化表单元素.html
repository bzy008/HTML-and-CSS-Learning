新的伪类 1.focus 元素聚焦的样式 2.checked 单选或多选框被选中的样式

性别：

常见用法 重置表单元素样式 input, textarea, button, select{ border:none; } input:focus, ...{ outline:none; outline-offset:none } 设置textarea是否允许调整尺寸 css属性resize： both：默认值 none：不能调整尺寸 horizontal：水平方向调整 vertical：竖直方向调整 文本框边缘到内容的距离 input{ padding：0 10px; } input{ text-indent: 1em; } 控制单选和多选的样式
